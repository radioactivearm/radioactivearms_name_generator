# radioactivearms_name_generator
This project saw the gathering and reformatting of forename and surname data. Then using that data a typer command line app was made with the purpose of generating a list of random names using commands of either 'boy' or 'girl'. This generated list is then amended to a master txt file incase you forget to write down one you really liked.
Sources of Data
https://www.nrscotland.gov.uk/statistics-and-data/names/#
https://www.census.gov/topics/population/genealogy/data.html

# How to set up
1. clone the repository to a chosen location on your computer.
2. Find the file and open either a terminal or git bash in that folder
3. input is structured as such: python main.py 'command'
4. the two commands are: boy or girl
5. so if you are looking for female names: python main.py girl
6. you can find the master txt file in this folder: Names.txt
